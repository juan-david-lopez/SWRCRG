Sistema de Microservicios

Arquitectura backend distribuida para la gestión integral de empleados. Despliegue continuo en la nube para alta disponibilidad.

Desplegado en Vercel

API Gateway

Operativo

Punto de entrada único para todos los clientes. Maneja el enrutamiento, balanceo de carga y validación inicial.

Node.js Express Port: 8085

Auth Service

Operativo

Emisión y validación de tokens JWT stateless. Gestión de roles, autenticación de usuarios y recuperación de credenciales.

Node.js PostgreSQL JWT

Empleados Service

Operativo

Microservicio core para la gestión del ciclo de vida del empleado (Onboarding, Offboarding, Reactivación).

Node.js PostgreSQL RabbitMQ

Departamentos Service

Operativo

Administración de áreas corporativas y estructuras organizacionales. Suministra validación a empleados.

Python FastAPI PostgreSQL

Perfiles Service

Operativo

Gestión de la información pública y configuraciones específicas del perfil del empleado.

Node.js PostgreSQL RabbitMQ

Notificaciones Service

Operativo

Envío asíncrono de correos electrónicos y notificaciones al detectar eventos del dominio.

Node.js RabbitMQ SMTP
